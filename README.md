# Supply Chain Analysis Using Python

## Overview

This project involved analysing supply chain data using Python to explore revenue, inventory, supplier performance, shipping operations and manufacturing efficiency. The objective was to identify key business trends and generate data-driven insights by cleaning, analysing and visualising the dataset.

## Skills Demonstrated

- Data Cleaning
- Data Analysis
- Exploratory Data Analysis (EDA)
- Data Visualisation
- Business Analysis
- Data Aggregation
- GroupBy
- Data Interpretation

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Visual Studio Code
- GitHub

## Conclusion

Through this analysis, I identified key trends across the supply chain dataset. The results showed that skincare products generated the highest revenue, whilst Carrier C recorded the highest average shipping costs and Carrier B the lowest. Transportation analysis highlighted differences in delivery performance across shipping methods. Inventory analysis identified product categories with higher stock levels. Supplier analysis revealed variations in defect rates and manufacturing costs showed little correlation with revenue generated. This project strengthened my ability to use Python to clean, analyse and visualise business data, transforming raw data into meaningful insights that support data-driven decision-making.
